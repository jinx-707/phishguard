🔄 DATA FLOW (Complete Integration)

1. User visits webpage
   ↓
2. Chrome Extension (content.js)
   - Extracts page data
   - Runs local AI inference (< 50ms)
   ↓
3. If suspicious → background.js
   - POST http://localhost:8000/scan
   ↓
4. FastAPI Server (app/main.py)
   - Receives request
   - Checks Redis cache
   ↓
5. If not cached:
   - ML Predictor analyzes content
   - Graph Service checks infrastructure
   - Scoring Engine combines results
   ↓
6. Store in PostgreSQL
   Cache in Redis
   ↓
7. Return JSON response
   ↓
8. Extension displays result:
   - HIGH risk → Full-screen block
   - MEDIUM risk → Warning banner
   - LOW risk → Allow

🛠️ TROUBLESHOOTING

Issue: Port 8000 already in use
Solution: netstat -ano | findstr :8000
          taskkill /PID <PID> /F

Issue: PostgreSQL not connecting
Solution: docker-compose restart db
          docker logs phishguard-postgres

Issue: Redis not connecting
Solution: docker-compose restart redis
          docker exec -it phishguard-redis redis-cli PING

Issue: Extension not loading
Solution: Check chrome://extensions/
          Verify folder path is correct
          Check for errors in console
