# Security Philosophy

PhishGuard is built on the belief that phishing is a human problem before it is a technical one.

Attackers exploit urgency, fear, authority, and trust. Our system is designed to identify
and neutralize these manipulation techniques before users are exposed to them.

We prioritize:
- Pre-interaction detection
- Explainable decisions
- Privacy-preserving, on-device intelligence
- Adaptation to evolving threat tactics
